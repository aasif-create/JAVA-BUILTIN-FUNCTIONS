# JAVA-BUILTIN-METHODS/FUNCTIONS

PRACTICED THE BUILT-IN-METHODS IN JAVA TO UNDERSTAND HOW DIFFERENT METHODS WORK AND HOW THEY CAN BE APPLIED IN REAL PROGRAMS.

# 1. STRING METHODS : Methods used to create, modify, compare, and process text in Java.
1. length()              -  Returns number of characters
2. charAt()              -  Returns character at a specific index
3. substring()           -  Extracts a part of the string
4. equals()              -  Compares two strings (case-sensitive)
5. equalsIgnoreCase()    -  Compares strings ignoring case
6. toLowerCase()         -  Converts string to lowercase
7. toUpperCase()         -  Converts string to uppercase
8. contains()            -  Checks if string contains a substring
9. replace()             -  Replaces characters or words
10. trim()               -  Removes extra spaces at ends
11. split()              -  Splits string using a separator

# 2. MATH METHODS : Methods used to perform mathematical calculations and numeric operations.
1. Math.abs()            -  Returns absolute positive value
2. Math.max()            -  Returns bigger of two numbers
3. Math.min()            -  Returns smaller of two numbers
4. Math.pow()            -  Returns a^b (power value)
5. Math.sqrt()           -  Returns square root
6. Math.random()         -  Returns random value (0â€“1)
7. Math.round()          -  Rounds to nearest integer
8. Math.ceil()           -  Rounds up to next integer
9. Math.floor()          -  Rounds down to previous integer

# 3. ARRAY UTILITY METHODS : Methods used to sort, search, convert, and process arrays easily.
1. Arrays.sort()         -  Sorts array in ascending order
2. Arrays.toString()     -  Converts array to string format
3. Arrays.equals()       -  Checks if two arrays are equal
4. Arrays.copyOf()       -  Copies array to new array
5. Arrays.fill()         -  Fills entire array with one value

# 4. COLLECTIONS UTILITY METHODS : Methods used to manage dynamic data structures like ArrayList, Set, and Map.
1. Collections.sort()    -  Sorts list in ascending order
2. Collections.reverse() -  Reverses the list order
3. Collections.shuffle() -  Randomly shuffles the list
4. Collections.max()     -  Returns largest element in list
5. Collections.min()     -  Returns smallest element in list

# 5. WRAPPER CLASS METHODS : Methods used to convert between primitives, objects, and strings.
1. Integer.parseInt()    -  Converts string to integer
2. Double.parseDouble()  -  Converts string to double
3. String.valueOf()      -  Converts any value to string
4. Integer.toString()    -  Converts integer to string
5. Double.toString()     -  Converts double to string
